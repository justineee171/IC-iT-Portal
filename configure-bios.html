Configure BIOS/UEFI Settings

Accessing BIOS/UEFI

To access BIOS/UEFI, restart your computer and press the designated key during startup (usually F2, DEL, or ESC). The exact key may vary by manufacturer and is typically shown on the screen during the boot process.

Understanding BIOS vs. UEFI

BIOS (Basic Input Output System) is the older firmware interface. UEFI (Unified Extensible Firmware Interface) is more modern, supports larger drives, faster booting, and has a graphical interface with mouse support.

Change Boot Order

Modify the boot priority to allow your system to boot from a USB drive or DVD. Move the installation media to the top of the boot list to install an operating system.

Enable Virtualization

Turn on virtualization (Intel VT-x or AMD-V) to support virtual machines and some server applications. This is usually under Advanced or CPU Settings.

Secure Boot Settings

Secure Boot helps prevent unauthorized OS from loading. Disable this setting only if your OS or drivers are not signed, especially during Linux installations.

SATA Mode Configuration

Set the SATA operation mode to AHCI or RAID as needed. AHCI is generally preferred for SSDs to enable full performance features like NCQ and hot-swapping.

Save and Exit

Once configuration is complete, save your changes and exit. Typically, this is done by pressing F10. The system will reboot using the updated settings.

Reset to Defaults

If misconfiguration occurs or issues arise, you can reset BIOS/UEFI to factory defaults. Look for an option like Load Default Settings or Reset to Optimized Defaults.
