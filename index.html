Welcome to IC-iT Portal

Your interactive learning resource for computer systems, networks, and more.

What is IC-iT?

IC-iT is a digital learning platform designed for ict students and college students pursuing Computer Systems Servicing. Whether you're getting started with hardware installation, setting up computer networks, configuring servers, or performing maintenance tasks, IC-iT provides structured lessons, real-world activities, and interactive content to help you build practical, job-ready skills.

How to Use This Portal

Explore the Lessons to access lessons grouped by learning outcomes. Each topic includes explanations, visuals, and hands-on activities. Use the navigation bar above to jump into any section you're interested in.
