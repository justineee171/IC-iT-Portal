Network Devices Overview

Router

A router connects multiple networks together and routes data packets between them. It is commonly used to connect a local area network (LAN) to the internet. Routers assign IP addresses, perform network address translation (NAT), and provide firewall functionality.

  • Distributes internet connection across devices
  • Supports both wired and wireless connections
  • Often includes basic firewall and DHCP services

Switch

Switches are networking devices that connect multiple devices within the same network. Unlike hubs, switches use MAC addresses to forward data only to the intended recipient device, improving network efficiency.

  • Operates at Layer 2 (Data Link Layer) of the OSI model
  • Supports full-duplex communication
  • Helps reduce network collisions and traffic

Hub

Hubs are basic networking devices that broadcast incoming data to all connected devices. They do not differentiate between devices, making them less secure and efficient than switches.

  • Operates at Layer 1 (Physical Layer)
  • No traffic management or filtering
  • Rarely used in modern networks due to inefficiency

Access Point (AP)

Access Points provide wireless connectivity to wired networks. They allow Wi-Fi enabled devices to connect to the network and are often integrated into routers for home use.

  • Extends wireless coverage within a building
  • Can be managed individually or via a centralized controller
  • Supports security protocols like WPA3

Modem

A modem (modulator-demodulator) converts digital signals to analog for transmission over telephone or cable lines and vice versa. It connects the local network to the internet via an ISP.

  • Essential for DSL, cable, or fiber internet connections
  • Can be standalone or combined with routers
  • Must be compatible with ISP standards

Firewall

A firewall is a network security device that monitors and controls incoming and outgoing traffic based on security rules. It can be hardware-based, software-based, or both.

  • Prevents unauthorized access to or from a private network
  • Can block specific IPs, ports, or protocols
  • Essential for securing sensitive networks
